\doxysubsection{Detailed Description}
This class represents a single cell of the Game Grid. 

This class is part of the \mbox{\hyperlink{classbridges_1_1_bridges}{Bridges}} Game API Each cell of the game grid contains background and foreground colors, and a symbol. Refer to the Named\+Color and Named\+Symbol definitions for a list of colors and symbols that can be used.
