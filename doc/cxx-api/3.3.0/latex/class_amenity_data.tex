\doxysubsection{Detailed Description}
Class that holds Open Street Map Amenity Data. 

This class holds the data for all the amenities requested by the user

Usually this class is not created by the user but returned by \mbox{\hyperlink{classbridges_1_1_data_source_a3ae31729bc8c4154dc091f89e9b7349e}{bridges\+::\+Data\+Source\+::get\+Amenity\+Data()}}
