\doxysubsection{Detailed Description}
Class that hold elevation data. 

Class that holds Elevation data.

A user would not normally create an \mbox{\hyperlink{classbridges_1_1dataset_1_1_elevation_data}{Elevation\+Data}} object but rather obtain one from calling \mbox{\hyperlink{classbridges_1_1_data_source_ac10b2fc8f5038adceb8e350be09ab52b}{bridges\+::\+Data\+Source\+::get\+Elevation\+Data()}}

A tutorial on how to use the Elevation dataset is available at\+: \href{https://bridgesuncc.github.io/tutorials/Data_Elevation.html}{\texttt{ https\+://bridgesuncc.\+github.\+io/tutorials/\+Data\+\_\+\+Elevation.\+html}}
