\doxysubsection{Detailed Description}
Class that hold Open Street Map Amenity data. 

This class holds the individual information for each amenity requested

Usually this class is not created by the user but part of an \mbox{\hyperlink{class_amenity_data}{Amenity\+Data}} object returned by \mbox{\hyperlink{classbridges_1_1_data_source_a3ae31729bc8c4154dc091f89e9b7349e}{bridges\+::\+Data\+Source\+::get\+Amenity\+Data()}}
