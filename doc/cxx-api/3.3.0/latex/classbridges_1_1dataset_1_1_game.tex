\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classbridges_1_1dataset_1_1_game}{Game}} object, used along with the Games data source. 

This is a convenience class provided for users who wish to use this data source as part of their application. It provides an API that makes it easy to access the attributes of this data set.

Each game record has title, platform on which it can be played, rating, and a list of genres.

One would not normally create an object of this type, but rather obtain them through a call to \mbox{\hyperlink{classbridges_1_1_data_source_aa902e1dffd4169e2f6e6820299b8cde1}{bridges\+::\+Data\+Source\+::get\+Game\+Data()}}.
