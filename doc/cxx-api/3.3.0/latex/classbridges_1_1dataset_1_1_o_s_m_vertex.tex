\doxysubsection{Detailed Description}
Class that hold Open Street Map vertices. 

Class that holds Open Street Map vertices from \href{https://openstreetmap.org}{\texttt{ https\+://openstreetmap.\+org}} Objects from this class are typically not created by the user but contained in the Osm\+Data object returned by \mbox{\hyperlink{classbridges_1_1_data_source_a57736934a90bdb86948a7c338ec81a2d}{bridges\+::\+Data\+Source\+::get\+OSMData()}}
