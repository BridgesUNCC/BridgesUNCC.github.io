\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classbridges_1_1dataset_1_1_song}{Song}} object, used along with the Songs data source. 

This is a convenience class provided for users who wish to use this data source as part of their application. It provides an API that makes it easy to access the attributes of this data set.

The \mbox{\hyperlink{classbridges_1_1dataset_1_1_song}{Song}} object is typically obtained from calling \mbox{\hyperlink{classbridges_1_1_data_source_a5e8d035a1becf96c71569e0966e93849}{bridges\+::\+Data\+Source\+::get\+Song\+Data()}} or \mbox{\hyperlink{classbridges_1_1_data_source_a7797e423d77ad467f9d98ebf78488c04}{bridges\+::\+Data\+Source\+::get\+Song()}}.
