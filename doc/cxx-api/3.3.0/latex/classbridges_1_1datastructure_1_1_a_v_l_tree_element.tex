\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename K, typename E$>$\newline
class bridges\+::datastructure\+::\+AVLTree\+Element$<$ K, E $>$}

This class can be used to create avl tree elements, derived from \mbox{\hyperlink{classbridges_1_1datastructure_1_1_b_s_t_element}{BSTElement}}. 

This class extends the \mbox{\hyperlink{classbridges_1_1datastructure_1_1_b_s_t_element}{BSTElement}} class by adding height and balance factor attributes to allow for easier use in a avl tree implementation.
