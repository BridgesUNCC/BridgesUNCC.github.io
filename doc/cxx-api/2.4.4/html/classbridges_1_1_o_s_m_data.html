◆ getGraph()

void bridges::OSMData::getGraph ( GraphAdjList< int, OSMVertex, double > *  gr) const
inline

Construct a graph out of the vertex an edge data of the OSM object. The graph will associate the length of the edge to the graph edge. No data is bound to the vertices.

The vertices of the graph will be located at the location where given in the data set converted to cartesian coordinate.
