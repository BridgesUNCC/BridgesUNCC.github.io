Detailed Description

template<typename Key, typename E>
class bridges::GraphAdjMatrix< Key, E >

This class provides methods to represent adjacency list based graphs.

The GraphAdjMatrix class can be used to represent adjacency matrix based graphs in BRIDGES, with Key representing a data specific generic parameter. The class is simply a wrapper around the C++ STL unordered_map class and, thus, derives all its operations from it. Given the use of operator overloading, the adjacency matrix implementation is almost identical to a normal array representation, except that any ordered type can be used to index into the matrix.

BRIDGES provides methods to visualize the graph and its contents.
