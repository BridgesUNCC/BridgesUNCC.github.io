Detailed Description

template<typename K, typename E>
class bridges::ADTVisualizer< K, E >

The ADTVisualizer class generates each data structure's JSON representation.

The ADTVisualizer class is responsible for taking a data structure as input and generating a JSON representation prior to sending it to the server for visualization. The Bridges class uses this object to keep track of the Visualization representation.

An end user will generally not need to interact directly with this class.
