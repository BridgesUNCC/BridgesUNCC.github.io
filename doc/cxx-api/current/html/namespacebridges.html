Detailed Description

This class can be used to create arrays of type Element<E> where E is a generic type representation application specific data. Arrays are internally represented as 1D arrays; currently 1D, 2D and 3D arrays are supported.

This is a class in BRIDGES for representing an (n x n) grid.

these methods convert byte arrays in to base64 codes and are used in BRIDGES to represent the color arrays as strings. The code is adapted from external sources detailed below.
