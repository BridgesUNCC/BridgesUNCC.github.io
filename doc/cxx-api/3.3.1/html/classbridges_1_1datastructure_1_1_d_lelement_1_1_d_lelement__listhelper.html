Detailed Description

template<typename E>
class bridges::datastructure::DLelement< E >::DLelement_listhelper

these are helper classes for DLelement for easy iteration in a range for loop. It is not meant to be created by the bridges user. But it may be returned by Bridges to provide an STL compliant list API.
