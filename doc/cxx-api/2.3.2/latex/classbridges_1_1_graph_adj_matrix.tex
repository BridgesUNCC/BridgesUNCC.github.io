\subsection{Detailed Description}
\subsubsection*{template$<$typename K, typename E$>$class bridges\+::\+Graph\+Adj\+Matrix$<$ K, E $>$}

This class provides methods to represent adjacency matrix based graphs. 

The class is simply a wrapper around the C++ S\+T\+L unordered\+\_\+map class and thus derives all its operations from it. Given the use of operator overloading, the adjacency matrix implementation is almost identical to a normal array representation, except that any ordered type can be used to index into the matrix.

Generic Parameters\+: K that is used as an index, E the application data type
