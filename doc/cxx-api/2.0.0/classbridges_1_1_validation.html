Detailed Description

This class performs certain validation checks to ensure the robustness of the BRIDGES system.

Validatoin checks are made for legal specification of visual properties such as color names, element size, opacity values, the maximum number of BRIDGES elements (currently set at 5000).

void bridges::Validation::validateColor ( string  color)

Determine if a color is supported by CSS.

This method only supports a subject of CSS (yet). (1) 173 CSS extended color names, (2) #RRGGBB or #RGB, where R, G and B are red, green, blue values as hexadecimal digits.

This method does not check for null because null has special meaning.
