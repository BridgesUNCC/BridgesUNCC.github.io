Detailed Description

template<typename E>
class bridges::Element< E >

This is the fundamental building block for all data structures in BRIDGES.

This is the Superclass Element with SLelement, DLelement, MLElement, CircSlElement, CircDlElement, TreeElement, AVLTreeElement, BSTElement subclasses.

Generic Parameters: E the application data type

The label field(string type) is used to label the visualization of the element.

Element holds a LinkVisualizer for each of its links and an ElementVisualizer for itself
