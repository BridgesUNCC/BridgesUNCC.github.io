bridges.data_src_dependent.elevation.ElevationData Class Reference

Detailed Description

Class that holds elevation data.

The data is stored as a 2d array .data of .cols columns and .rows rows. The data at [0][0] are at location xll, yll and the spatial resolution is cellsize.

This object is generally not created by the user, to see how its created check out bridges::data_src_dependent::data_source::get_elevation_data()

See also
For an example, check out https://bridgesuncc.github.io/tutorials/Data_Elevation.html
