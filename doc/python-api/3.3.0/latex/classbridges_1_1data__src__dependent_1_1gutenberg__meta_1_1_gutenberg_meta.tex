\doxysubsection{Detailed Description}
Class to hold the meta data of various books from the Gutenberg Project. 

This is to be used with the Gutenberg request functions.

This class holds id, title, language, date\+\_\+added, authors, genres, and library of congress classifications.

This object is generally not created by the user, to see how its created check out \mbox{\hyperlink{namespacebridges_1_1data__src__dependent_1_1data__source_aa2cf9b93ab39a3f721068be1b99a4000}{bridges\+::data\+\_\+src\+\_\+dependent\+::data\+\_\+source\+::get\+\_\+gutenberg\+\_\+book\+\_\+metadata()}}
