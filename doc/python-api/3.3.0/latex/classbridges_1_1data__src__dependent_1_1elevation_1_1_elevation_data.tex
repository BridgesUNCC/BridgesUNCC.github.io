\doxysubsection{Detailed Description}
Class that holds elevation data. 

The data is stored as a 2d array .data of .cols columns and .rows rows. The data at \mbox{[}0\mbox{]}\mbox{[}0\mbox{]} are at location xll, yll and the spatial resolution is cellsize.

This object is generally not created by the user, to see how its created check out \mbox{\hyperlink{namespacebridges_1_1data__src__dependent_1_1data__source_ac6662947ed387a2304495eb53914bba3}{bridges\+::data\+\_\+src\+\_\+dependent\+::data\+\_\+source\+::get\+\_\+elevation\+\_\+data()}}
