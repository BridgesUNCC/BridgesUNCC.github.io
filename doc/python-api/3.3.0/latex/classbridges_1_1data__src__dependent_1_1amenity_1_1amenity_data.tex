\doxysubsection{Detailed Description}
This class is a helper class to be used with amenities data retrieved from Open\+Street Map data. 

The dataset contains amenity id, latitude, longitude, name, and a list of various other values.

This object is generally not created by the user, to see how its created check out \mbox{\hyperlink{namespacebridges_1_1data__src__dependent_1_1data__source_a2e0450aba02ebf02bb48c84ea0f1b516}{bridges\+::data\+\_\+src\+\_\+dependent\+::data\+\_\+source\+::get\+\_\+amenity\+\_\+data()}}
