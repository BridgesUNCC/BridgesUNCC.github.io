\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classbridges_1_1data__src__dependent_1_1song_1_1_song}{Song}} object, used along with the Songs data source. 

This is a convenience class provided for users who wish to use this data source as part of their application. It provides an API that makes it easy to access the attributes of this data set.

This object is generally not created by the user, to see how its created check out \mbox{\hyperlink{namespacebridges_1_1data__src__dependent_1_1data__source_a060ab8ec1777a5458a37d5e01e594e82}{bridges\+::data\+\_\+src\+\_\+dependent\+::data\+\_\+source\+::get\+\_\+song()}}
