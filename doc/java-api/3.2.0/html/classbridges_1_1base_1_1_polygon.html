Detailed Description

This class defines a polygon and is part of the symbol collection. A polygon has a sequence of 2D points (x, y coordinate pairs)

Basic styling such as stroke, fill, color are defined in the superclass Symbol.

See also
An example tutorial can be found at http://bridgesuncc.github.io/tutorials/Symbol_Collection.html
