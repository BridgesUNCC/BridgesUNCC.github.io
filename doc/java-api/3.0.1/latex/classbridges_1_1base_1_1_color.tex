\subsection{Detailed Description}
\begin{DoxyVerb}@brief This class is used to represent colors in BRIDGES.

We use an RGBA model
to represent colors, with each component in the range 0-255. BRIDGES
also supports named colors for user convenience, but these are converted
into [RGBA] prior to transmission to the server for visualization.

<p>
\end{DoxyVerb}


Supported named colors are listed below and here\+: \href{https://drafts.csswg.org/css-color-3/\#svg-color}{\texttt{ https\+://drafts.\+csswg.\+org/css-\/color-\/3/\#svg-\/color}}
