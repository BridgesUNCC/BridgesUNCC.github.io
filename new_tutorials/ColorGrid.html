Color Grid Tutorial


Grid<E> implements a 2D grid structure in BRIDGES, and it can be used to represent 2D image like structures (more efficient than the 2D array representation for large arrays), as well as implementation of simple 2D games that operate on moderate sized grids. ColorGrid is derived from Grid<E> where E is replaced by the Color type. The setDimensions() method is used to set the grid dimensions. The ColorGrid subclass specializes this class to use Color for each grid element and is the subject of the tutorial here. As before, <E> represents the generic class parameter. The Grid type uses a base64 representation for its contents.

Look at the following classes that are directly or indirectly used in this tutorial:

  1. Grid
  2. ColorGrid
  3. Color

This tutorial will illustrate the use of Color Grid in BRIDGES. The tutorial consists of 3 parts:

  1. A basic tutorial on how to create a color grid and draws a Plus sign in different colors
  2. How to style the color grid elements with various visual attributes
  3. Demonstrates an algorithm to construct a checkerboard pattern
