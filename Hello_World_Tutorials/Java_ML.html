Create a new .java file

Imports

  • We need to include these Bridges files to give access to all the classes/methods needed to interact with Bridges
  • In your .java file, enter the following code snippets:
    •       import bridges.connect.Bridges;
      import bridges.base.MLelement;

Main Exception

  • By adding a throw exception we can forgo messy try/catch blocks in our code for our Bridges calls
  • In your .java file, add a throw exception to the main function so it looks like this:
    • public static void main(String[] args) throws Exception

Inside our Main

  • First we need to create our BRIDGES object and initialize our BRIDGES Credentials; here we use for our example a StudentInfo object class, which will be a generic parameter.
    • Bridges<String, String> 
	bridges = new Bridges(1, "YOUR_API_KEY", "YOUR_USER_ID");
    In the above call, the parameter 1 is the assignment id (can be any positive integer), and the remaining two parameters are your BRIDGES credentials. You will need to replace the two fields in this call with your own BRIDGES credentials.

  • Then we can create our MLelements; here we will illustrate a simple multilist appliciation with strings, with sublists from 2 nodes
    • First, create the main list with 5 nodes and link them; this is just a normal singly linked list. 
      • 	MLelement<String> el0 = new MLelement<String>("0", "0");
	MLelement<String> el1 = new MLelement<String>("1", "1");
	MLelement<String> el2 = new MLelement<String>("2", "2");
	MLelement<String> el3 = new MLelement<String>("3", "3");
	MLelement<String> el4 = new MLelement<String>("4", "4");
							// link them
	el0.setNext(el1);
	el1.setNext(el2);
	el2.setNext(el3);
	el3.setNext(el4);
    • Next create a sublist of 3 elements and make it a sublist of node 2
      • 							// create a sublist from node 2
	MLelement<String> el5 = new MLelement<String>("5", "5");
	MLelement<String> el6 = new MLelement<String>("6", "6");
	MLelement<String> el7 = new MLelement<String>("7", "7");
							// link the sublist
	el5.setNext(el6);
	el6.setNext(el7);
	el2.setSubList(el5);
    • Next, create a second sublist of 2 nodes from node 3 
      • 							// create a second sublist from 3
	MLelement<String> el8 = new MLelement<String>("8", "8");
	MLelement<String> el9 = new MLelement<String>("9", "9");

							// link the sublist
	el8.setNext(el9);
	el3.setSubList(el8);
    • Finally, create sublist of 2 elements and make it a sublist of the sublist from node 3! 
      • 							// create a (sub)sublist from 5
	MLelement<String> el10 = new MLelement<String>("10", "10");
	MLelement<String> el11 = new MLelement<String>("11", "11");
							// link the sublist
	el10.setNext(el11);
	el6.setSubList(el10);
    • Set some visual attributes to teh nodes 
      • 							// set visual attributes
	el0.getVisualizer().setSize (40);
	el0.getLinkVisualizer(el1).setColor("red");
	el0.getLinkVisualizer(el1).setThickness(3);
	el1.getLinkVisualizer(el2).setColor("blue");
	el1.getLinkVisualizer(el2).setThickness(3);
	el2.getLinkVisualizer(el3).setColor("cyan");
	el2.getLinkVisualizer(el3).setThickness(3);
	el0.getVisualizer().setColor("red");
    • Give Bridges the data structure handle 
      • 		bridges.setDataStructure(el0);
    • Visualize the multi list
      • 		bridges.visualize();

      Summary. Here's the final code:

      
import bridges.connect.Bridges;
import bridges.base.MLelement;

public class mllist {
	public static void main(String[] args) throws Exception{
    	
    								//create the Bridges object
		Bridges bridges = new Bridges(33, "YOUR API KEY", "YOUR USER ID");
        
		bridges.setTitle("A Multilist Example");

   							//create  nodes for the main list

		MLelement<String> el0 = new MLelement<String>("0", "0");
		MLelement<String> el1 = new MLelement<String>("1", "1");
		MLelement<String> el2 = new MLelement<String>("2", "2");
		MLelement<String> el3 = new MLelement<String>("3", "3");
		MLelement<String> el4 = new MLelement<String>("4", "4");

							// link them
		el0.setNext(el1);
		el1.setNext(el2);
		el2.setNext(el3);
		el3.setNext(el4);
							// create a sublist from node 2
		MLelement<String> el5 = new MLelement<String>("5", "5");
		MLelement<String> el6 = new MLelement<String>("6", "6");
		MLelement<String> el7 = new MLelement<String>("7", "7");

							// link the sublist
		el5.setNext(el6);
		el6.setNext(el7);
		el2.setSubList(el5);

							// create a second sublist from 3
		MLelement<String> el8 = new MLelement<String>("8", "8");
		MLelement<String> el9 = new MLelement<String>("9", "9");

							// link the sublist
		el8.setNext(el9);
		el3.setSubList(el8);

							// create a (sub)sublist from 5
		MLelement<String> el10 = new MLelement<String>("10", "10");
		MLelement<String> el11 = new MLelement<String>("11", "11");

							// link the sublist
		el10.setNext(el11);
		el6.setSubList(el10);

							// set visual attributes
		el0.getVisualizer().setSize (40);
		el0.getLinkVisualizer(el1).setColor("red");
		el0.getLinkVisualizer(el1).setThickness(3);
		el1.getLinkVisualizer(el2).setColor("blue");
		el1.getLinkVisualizer(el2).setThickness(3);
		el2.getLinkVisualizer(el3).setColor("cyan");
		el2.getLinkVisualizer(el3).setThickness(3);

		el0.getVisualizer().setColor("red");
             						//set visualizer type
		bridges.setDataStructure(el0);
        							// visualize the multilist
		bridges.visualize();
    }
}
