Create a new .java file

Imports

  • We need to include these Bridges files to give access to all the classes/methods needed to interact with Bridges
  • In your .java file, enter the following code snippets:
    • import bridges.connect.Bridges; import bridges.base.BSTElement;

Main Exception

  • By adding a throw exception we can forgo messy try/catch blocks in our code for our Bridges calls
  • In your .java file, add a throw exception to the main function so it looks like this:
    • public static void main(String[] args) throws Exception

Inside our Main

  • First we need to create our Bridges object and initialize our Bridges Credentials
    • Bridges<Integer, String> bridges = new Bridges<>(4, "YOUR_API_KEY", "YOUR_USER_ID");
  • Then we can create our AVL Tree elements; here we will show a simple example of an AVL tree manually created with balance factors manually put in; in a real project, students will implement insert/delete/find operations on the tree that will automatically maintain these balance factors after AVL tree rotations.
    • AVLTreeElement avl_root = new AVLTreeElement(50, "Hello"); AVLTreeElement avl_2 = new AVLTreeElement(30, "This"); AVLTreeElement avl_3 = new AVLTreeElement(70, "is"); AVLTreeElement avl_4 = new AVLTreeElement(20, "an"); AVLTreeElement avl_5 = new AVLTreeElement(40, "AVL"); AVLTreeElement avl_6 = new AVLTreeElement(90, "Tree");
  • Now set the balance factors as the labels of the nodes, so we can see them in the BRIDGES visualization by mousing over the nodes
    • avl_root.setLabel("0"); avl_2.setLabel("0"); avl_3.setLabel("-1"); avl_4.setLabel("0"); avl_5.setLabel("0"); avl_6.setLabel("0");
  • Now link the AVL Tree elements to build the tree
    • avl_root.setLeft(avl_2); avl_root.setRight(avl_3); avl_2.setLeft(avl_4); avl_2.setRight(avl_5); avl_3.setRight(avl_6);
  • Now we pass the root of our AVL Tree to Bridges
    • bridges.setDataStructure(avl_root);
  • Finally we visualize the tree
    • bridges.visualize();

import bridges.connect.Bridges; import bridges.data_src_dependent.Tweet; import bridges.base.BSTElement; public class avlDriver { public static void main(String[] args) throws Exception{ //create the Bridges object Bridges bridges = new Bridges(15, "YOU_API_KEY", "YOUR_USER_ID"); //create an AVL tree AVLTreeElement avl_root = new AVLTreeElement(50, "Hello"); AVLTreeElement avl_2 = new AVLTreeElement(30, "This"); AVLTreeElement avl_3 = new AVLTreeElement(70, "is"); AVLTreeElement avl_4 = new AVLTreeElement(20, "an"); AVLTreeElement avl_5 = new AVLTreeElement(40, "AVL"); AVLTreeElement avl_6 = new AVLTreeElement(90, "Tree"); // put in the balance factors - done manually here // for illustration, but in a real project, the user will // implement this as part of the insert/delete // operations on the tree to maintain a balanced tree avl_root.setLabel("0"); avl_2.setLabel("0"); avl_3.setLabel("-1"); avl_4.setLabel("0"); avl_5.setLabel("0"); avl_6.setLabel("0"); avl_root.setLeft(avl_2); avl_root.setRight(avl_3); avl_2.setLeft(avl_4); avl_2.setRight(avl_5); avl_3.setRight(avl_6); // set some attributes avl_root.getVisualizer().setColor("cyan"); avl_root.getVisualizer().setOpacity(0.1); //set visualizer type bridges.setDataStructure(avl_root); // visualize the list bridges.visualize(); } }
