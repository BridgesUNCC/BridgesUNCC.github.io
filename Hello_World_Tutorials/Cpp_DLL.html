Create a new .cpp file

Includes

  • We need to include these Bridges headers to give access to all the classes/methods needed to interact with Bridges
  • In your .cpp file, enter the following code snippets:
    • #include "DLelement.h" #include "Bridges.h"

Namespace

  • By using this namespace we can forgo messy scope specifiers in our code for our Bridges calls
  • In your .cpp file, enter the following code snippet:
    • using namespace bridges;

Inside our Main

  • First we need to initialize our Bridges Credentials
    • Bridges::initialize(2, "YOUR_API_KEY","YOUR_USER_ID");

    Note that you must replace the two strings above with your BRIDGES credentials.

  • Then we can create our DLelements
    • DLelement<int> e0(0,"Hello"); DLelement<int> e1(0,"World"); DLelement<int> e2(0, "!");
  • Now link the DLelements together
    • e0.setNext(&e1); e1.setNext(&e2); e2.setPrev(&e1); e1.setPrev(&e0);
  • Now we pass the first element of the doubly linked list to Bridges
    • Bridges::setDataStructure(&e0);
  • Finnaly we call the visualize function
    • Bridges::visualize();

Code Summary: Your .cpp file should look like this

#include "DLelement.h" #include "Bridges.h" using namespace bridges; int main() { //create the Bridges object Bridges::initialize("YOUR_API_KEY","YOUR_USER_ID",2); //create elements DLelement<int> e0(0,"Hello"); DLelement<int> e1(0,"World"); DLelement<int> e2(0, "!"); //link elements e0.setNext(&e1); e1.setNext(&e2); e2.setPrev(&e1); e1.setPrev(&e0); //pass first element of data structure Bridges::setDataStructure(&e0); //visualize data structure Bridges::visualize(); }
