Create a new .java file

Imports

  • We need to include these Bridges files to give access to all the classes/methods needed to interact with Bridges
  • In your .java file, enter the following code snippets:
    • import bridges.connect.Bridges; import bridges.base.BSTElement;

Main Exception

  • By adding a throw exception we can forgo messy try/catch blocks in our code for our Bridges calls
  • In your .java file, add a throw exception to the main function so it looks like this:
    • public static void main(String[] args) throws Exception

Inside our Main

  • First we need to create our Bridges object and initialize our Bridges Credentials
    • Bridges<Integer, String> bridges = new Bridges<>(4, "YOUR_API_KEY", "YOUR_USER_ID");
  • Then we can create our TreeElements
    • BSTElement<Integer,String> e0 = new BSTElement<>("",0, ""); BSTElement<Integer,String> e1 = new BSTElement<>("Hello",-1, ""); BSTElement<Integer,String> e2 = new BSTElement<>("World",1, "");
  • Now link the BSTElements together
    • e0.setLeft(e1); e0.setRight(e2);
  • Now we pass the root of our Binary Search Tree to Bridges
    • bridges.setDataStructure(e0);
  • Finnaly we visualize the tree
    • bridges.visualize();

Code Summary: Your .java file should look like this

import bridges.connect.Bridges; import bridges.base.BSTElement; public class HelloWorld { public static void main(String[] args) throws Exception { //create the Bridges object Bridges<Integer, String> bridges = new Bridges<>(4, "YOUR_API_KEY", "YOUR_USER_ID"); //create elements BSTElement<Integer,String> e0 = new BSTElement<>("",0, ""); BSTElement<Integer,String> e1 = new BSTElement<>("Hello",-1, ""); BSTElement<Integer,String> e2 = new BSTElement<>("World",1, ""); //link elements e0.setLeft(e1); e0.setRight(e2); //pass first element of data structure bridges.setDataStructure(e0); //visualize data structure bridges.visualize(); } }
