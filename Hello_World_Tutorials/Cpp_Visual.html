Create an Element

    Element<string> e;

Visualization changing

  • For C++, Bridges represents colors with its own color class
  • The class supports, rgb, rgba, hex, and named color values
  • Color can be changed like this:
    • e.setColor(Color(0,0,0,128));
  • Size can be changed like this:
    • e.setSize(20);
  • Shape can be changed like this:
    • e.setShape(DIAMOND);
