Create an Element

    Element<String> e = new Element<>;

Visualization changing

  • For Java, Bridges stores visualization properties in an ElementVisualizer class owned by each element.
  • So, to get change each property we must retrieve the element's ElementVisualizer.
  • Color can be changed like this:
    • e.getVisualizer().setColor("black");
  • For Java, Bridges handles opacity seperately from color.
  • Opacity can be changed like this:
    • e.getVisualizer().setOpacity(.5);
  • Size can be changed like this:
    • e.getVisualizer().setSize(20);
  • Shape can be changed like this:
    • e.getVisualizer().setShape("diamond");
