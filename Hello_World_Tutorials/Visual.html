Changing Element visualization properties

Each Element contains properties that control how its visualization looks. You can change how an Element is visualized by interacting with these properties. You can use them to change the size, shape, and color of each Element.

Changing Edge visualization properties

Each Element also contains a LinkVisualizer class that controls how the edges between the nodes appear. You can change how an edge is visualized by interacting with that Element's LinkVisualizer. You can use the LinkVisualizer to change the thickness and color of each edge:

Note that because Element is the superclass of all other element types, that these same methods will also apply to them as well.
