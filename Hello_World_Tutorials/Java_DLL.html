Create a new .java file

Imports

  • We need to include these Bridges files to give access to all the classes/methods needed to interact with Bridges
  • In your .java file, enter the following code snippets:
    • import bridges.connect.Bridges; import bridges.base.DLelement;

Main Exception

  • By adding a throw exception we can forgo messy try/catch blocks in our code for our Bridges calls
  • In your .java file, add a throw exception to the main function so it looks like this:
    • public static void main(String[] args) throws Exception

Inside our Main

  • First we need to create our Bridges object and initialize our Bridges Credentials
    • Bridges<String, String> bridge = new Bridges<>(2, "YOUR_API_KEY", "YOUR_USER_ID");

    Note that you must replace the above strings with your BRIDGES credentials.

  • Then we can create our DLelements
    • DLelement<String> e0 = new DLelement<>("Hello", ""); DLelement<String> e1 = new DLelement<>("World", ""); DLelement<String> e2 = new DLelement<>("!", "");
  • Now link the DLelements together
    • e0.setNext(e1); e1.setNext(e2); e2.setPrev(e1); e1.setPrev(e0);
  • Now we pass the first element of our datastruture to Bridges
    • bridge.setDataStructure(e0);
  • Finnaly we call the visualize function
    • bridge.visualize();

Code Summary: Your .java file should look like this

import bridges.connect.Bridges; import bridges.base.DLelement; public class HelloWorld { public static void main(String[] args) throws Exception { //create the Bridges object Bridges<String, String> bridge = new Bridges<>(2, "YOUR_API_KEY", "YOUR_USER_ID"); //create elements DLelement<String> e0 = new DLelement<>("Hello", ""); DLelement<String> e1 = new DLelement<>("World", ""); DLelement<String> e2 = new DLelement<>("!", ""); //link elements e0.setNext(e1); e1.setNext(e2); e2.setPrev(e1); e1.setPrev(e0); //pass first element of data structure bridge.setDataStructure(e0); //visualize data structure bridge.visualize(); } }
