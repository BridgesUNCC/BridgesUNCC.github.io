Create a new .java file

Imports

  • We need to include these Bridges files to give access to all the classes/methods needed to interact with Bridges
  • In your .java file, enter the following code snippets:
    • import bridges.connect.Bridges; import bridges.base.TreeElement;

Main Exception

  • By adding a throw exception we can forgo messy try/catch blocks in our code for our Bridges calls
  • In your .java file, add a throw exception to the main function so it looks like this:
    • public static void main(String[] args) throws Exception

Inside our Main

  • First we need to create our Bridges object and initialize our Bridges Credentials
    • Bridges<String> bridge = new Bridges<>(3, "YOUR_API_KEY", "YOUR_USER_ID");
  • Then we can create our TreeElements
    • TreeElement<String> e0 = new TreeElement<>("", ""); TreeElement<String> e1 = new TreeElement<>("Hello", ""); TreeElement<String> e2 = new TreeElement<>("World", "");
  • Now link the TreeElements together
    • e0.setLeft(e1); e0.setRight(e2);
  • Now we pass the first element of our datastruture to Bridges
    • bridge.setDataStructure(e0);
  • Finnaly we call the visualize function
    • bridge.visualize();

Code Summary: Your .java file should look like this

import bridges.connect.Bridges; import bridges.base.TreeElement; public class HelloWorld { public static void main(String[] args) throws Exception { //create the Bridges object Bridges<String> bridge = new Bridges<>(3, "YOUR_API_KEY", "YOUR_USER_ID"); //create elements TreeElement<String> e0 = new TreeElement<>("", ""); TreeElement<String> e1 = new TreeElement<>("Hello", ""); TreeElement<String> e2 = new TreeElement<>("World", ""); //link elements e0.setLeft(e1); e0.setRight(e2); //pass first element of data structure bridge.setDataStructure(e0); //visualize data structure bridge.visualize(); } }
