USGS Earthquake Data

This data set is extracted as Tweets from the US Geological Survey (USGS). The BRIDGES API extracts the quake data records periodically and retains the most recent 5000 quakes in a database. These are accessed by the BRIDGES client, reformatted into a list of objects for use in assignments.
