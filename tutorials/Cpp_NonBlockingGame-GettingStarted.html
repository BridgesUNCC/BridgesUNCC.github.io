C++ Getting Started

Assuming that your environment is already set to use Bridges, there is no additional setup steps to perform. (If not, look at the Getting Started page for your environment, or contact your instructor.)

Basically, the code need to be linked against curl, pthreads, and libbridges.a. libbridges.a is provided in the package for your operating system.

Note that the windows version of BRIDGES in C++ does not currently support games.
