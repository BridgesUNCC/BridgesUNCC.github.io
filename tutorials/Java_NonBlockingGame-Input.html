Create a InputGame.java file and copy the following code in.






Check out the documentation
  of NonBlockingGame
  for more details. In particular look at the
  functions setBGColor()
  and drawSymbol().



The list of colors available come
  from NamedColor
  and the list of symbols come
  from NamedSymbol.



The functions for accessing the keys are:
  keyUp(),
  keyDown(),
  keyLeft(),
  keyRight(),
  keyW(),
  keyA(),
  keyS(),
  keyD(),
  keyQ(),
  keySpace()
