US Cities Dataset

BRIDGES provides easy access to US Cities data. Each record consists of city ID, name, state, country, latitude, longitude, elevation and time zone. The API enables to filter the cities based on location, population, or elevation.

The program snippet below illustrates how to query for the dataset using some of the available filters.
