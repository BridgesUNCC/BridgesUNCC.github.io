US Cities Dataset

BRIDGES provides easy access to US Cities data. Each record consists of city ID, name, state, country, latitude, longitude, elevation and time zone. This data can be very useful in algorithms that can take advantage o location information, for instance geometric algorithms (convex hull, voronoi diagram, traveling salesman), graph algorithms (Minimum Spanning tree, shotest path, etc. For earlier courses such as CS1, CS2, we can plot the cities and use a variety of filtering options to customize the displayed city list.
