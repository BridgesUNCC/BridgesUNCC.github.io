Guttenberg Book Collection MetaData

Metadata from the top 1000 books (by download) from Project Gutenberg. This dataset is adapted from the Classics CSV Library, part of the CORGIS Dataset Project at Virginia Tech. This data set represents meta data of 1000 books in the Guttenberg collection. Each book record contains the title, author, author's birth and death dates, languages, subjects, number of words, number of difficult words, number of sentences and number of characters.
