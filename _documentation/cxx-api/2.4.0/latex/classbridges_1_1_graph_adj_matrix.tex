\subsection{Detailed Description}
\subsubsection*{template$<$typename K, typename E1 = K, typename E2 = E1$>$class bridges\+::\+Graph\+Adj\+Matrix$<$ K, E1, E2 $>$}

This class provides methods to represent adjacency matrix based graphs. 

The class is simply a wrapper around the C++ S\+T\+L unordered\+\_\+map class and thus derives all its operations from it. Given the use of operator overloading, the adjacency matrix implementation is almost identical to a normal array representation, except that any ordered type can be used to index into the matrix.

Since the adjacency matrix typically contains only a numerical value, we keep edge specific information in a separate map using a generic parameter for the type.

Generic Parameters\+: K that is used as an index, E1 information specific to graph vertices E2 information specific to graph edges
