Using Discord

We will use Discord as the mechanism to talk and see each others screen. Here are the steps you'll need to take:

  1. Install the Discord desktop app on your machine from here. Note that the is a web client to Discord but it is somewhat unreliable. The desktop app is much more reliable.

  2. Join our BRIDGES Discord server https://discord.gg/nzWN3x using the Discord desktop app.

    1. In the discord app, hit the "add a server button", it is the + on the left bar
    2. A pop up will appear, select "join a server"
    3. Paste the link to the BRIDGES Discord server https://discord.gg/nzWN3x

  3. Join the General voice channel by clicking on "General".

  4. Ask on the #help-needed text channel if you have issues with Discord or anything!
