Assignment 1 - List IMDB

Goals

The purpose of this assignment is to learn to 1. Access remote data through BRIDGES. 2. Manipulate a linked list. You will generate a visualization that looks like this:

Programming part

Task

Build a linked list containing each (actor, movie) pair that appears in the Actor Movie IMDB data set.

Steps

  1. Open your base code.
  2. Plug in your credentials.

  3. Compile and run the code and observe the basic linked list.

    Build Actor Movie linked list

  4. Change SLelement to be a list of ActorMovieIMDB.

  5. For each entry in the set of actor movie, create an SLelement to store it with an explicit label.
  6. We recommend you add each new entry to the head of the list for simplicity.
  7. Compile and run the code and observe the linked list of (actor,movie) pairs.
