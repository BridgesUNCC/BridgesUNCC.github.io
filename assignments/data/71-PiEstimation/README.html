Assignment 71 - Pi Estimation

## Goals

The purpose of this assignment is to learn to 1. Use a color grid to estimate the value of PI 2. Manipulate a ColorGrid object 3. Learn how to set/change colors for each pixel in the grid 4. Practice using simple loops to display the value of PI using the BRIDGES ColorGrid data structure

## Programming

Tasks

  • Initialize a random number generator
  • Generate points and check if they fall within the sqaure or circle
  • Calculate an estimation of pi using the Monte Carlo simulation method
